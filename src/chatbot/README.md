# `API 설치`
------------

+ ## `openai 설치`

    + pip install openai

+ ## `spaCy 설치`

    + pip install spacy


    ### spaCy 한글 모델 'ko_core_news_lg' 설치

    + python -m spacy download ko_core_news_lg

+ ## `konply 설치`

    + pip install konply

    ### konply 사용하려면 jdk 17 이상 버전이 필요함

+ ## `dotenv 설치`

    + pip install dotenv
